# ReactGram Frontend

This project was developed during the "React do Zero a Maestria" course on Udemy.

## Technologies and tools used

- React
- React Router DOM
- Redux Toolkit
- React Redux
- React Icons
- Vite
- JavaScript
- HTML
- CSS

## Features

- User registration
- Login and authentication
- Image upload
- Post feed
- Likes
- Comments
- User profile
- User search
- Profile editing
